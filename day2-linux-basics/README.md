# day2-linux-basics

## 목표
- 리눅스 파일시스템 구조 (`/etc`, `/proc`, `/home`)
- 프로세스 확인 (`ps`, `top`, `kill`)
- 권한 제어 (`chmod`, `sudo`)

## 실습 파일
- basic_syntax.c
- memory_layout.c

## 실행 방법
| `ls -al` | 숨김 파일 포함 상세 보기 |
| `grep` | 문자열 검색 |
| `find` | 파일 찾기 |
| `chmod` | 파일 권한 변경 |
| `ps aux` | 실행 중인 프로세스 보기 |
| `nano` | 파일 생성 |
| `gcc -o <실행파일>` | 실행 파일 생성 |
| `./<실행파일>` | 파일 실행 |


## 결과 요약
- `/etc/passwd` 구조 파악 완료
- `/proc` 통해 시스템 정보 접근 성공
- 권한 변경 및 sudo 실습 완료
